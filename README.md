### 🧠 Project 1 — Private AI Agent: Automated Test Case Generator

> **Stack:** `Ollama` · `AnythingLLM` · `RAG` · `Local LLMs` · `Python`

| | |
|---|---|
| **🔴 Problem** | Enterprise QA teams waste 60%+ of sprint time manually writing test cases from dense requirement documents — slow, inconsistent, and a data privacy risk when using cloud AI tools. |
| **🟡 Solution** | Architected and deployed a **100% offline, local-first AI Agent** using Ollama and RAG pipelines that ingests raw PDFs/Docs and auto-generates structured test suites (Positive, Negative, Edge-case scenarios). Zero cloud dependency = zero IP exposure. |
| **🟢 Result** | ⬇️ **60% reduction** in manual test drafting effort · ✅ **90%+ scenario accuracy** · 🔒 **100% data privacy** maintained within the local network · Team freed to focus on high-value exploratory testing. |
